Descrição geral:

Uma aplicação Back-End desenvolvida em Java.

Estrutura do projeto preparada com Spring Boot (pelo pom.xml e padrão de pastas src).

Inclui configuração para Docker, permitindo containerização do aplicativo.

Código organizado com controle de versão Git e suporte a build via Maven (mvnw presente).

Tecnologias utilizadas:

Java (principal linguagem, ~98% do código)

Spring Boot (para estrutura de API e lógica de negócios)

Docker (2% do projeto, provavelmente para deployment)

Maven (gerenciamento de dependências e build)

Funcionalidades prováveis:

CRUD completo para gerenciamento de tarefas (típico de projetos To‑Do List)

Possível persistência de dados via JPA/Hibernate, dado o uso de Spring Boot

Preparado para execução local e deployment em container Docker

Destaques do projeto:

Organização profissional de pastas e arquivos, seguindo padrões de projeto Java + Spring Boot

Integração com Docker, demonstrando conhecimentos em DevOps básicos

Uso de Maven e wrapper (mvnw) para facilitar execução do projeto em qualquer máquina
